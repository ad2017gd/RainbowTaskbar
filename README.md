# Getting started
First off, grab yourself the hottest new release at https://github.com/ad2017gd/RainbowTaskbar/releases.

RainbowTaskbar is a portable app, so there's no installing required.
If you do not already have it installed, the app will prompt you to install **.NET 6**.

After starting the app for the first time, you will automatically get the Rainbow preset. To open the editor or choose another preset, locate
the tray icon on your taskbar.

![](https://ad2017.dev/rnb/assets/tray.png)

**Have fun using RainbowTaskbar!**
